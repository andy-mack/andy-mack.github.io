How to switch line separators in Intellij Idea

When git shows changes in all you class or another file on commit, but you make changes in one or two lines, problem can be in line endings. Of course it is better to keep all code in same format. However sometimes it maybe required to show functional changes and you need to keep line separators as it in originally commited class.
